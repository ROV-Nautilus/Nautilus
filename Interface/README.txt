Le projet Nautilus a été programmé et compilé avec Eclipse Oxigen2
sous Java8 avec des librairies supplémentaires tel que jsch(fournit)
et java3d (a installer).

Nous avons laissé tous les images jpg utilisées ainsi que les images originales.

Le code est largement commenté dans son ensemble mais pas non plus ligne par ligne,
cela requiert queqlues connaissance en base de reseau et en 3d.
